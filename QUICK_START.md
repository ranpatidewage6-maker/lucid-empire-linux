## 3 Steps to Launch

### Step 1: Launch Control Panel
```
Double-click: LAUNCH_INSTALLER.bat
```

### Step 2: Select or Generate Profile
- **Option A:** Select existing profile from dropdown
- **Option B:** Generate new profile:
  1. Enter profile name
  2. Select persona (student/worker/gamer/professional)
  3. Set age (30-180 days)
  4. Click "GENERATE AGED PROFILE"

### Step 3: Launch Browser
```
Click: [ ENTER OBLIVION ]
```

Camoufox browser opens with your aged profile loaded. You have full control.

---

## First Time Setup

If this is your first time:

1. Launch `LAUNCH_INSTALLER.bat`
2. Click **INSTALL DEPENDENCIES** (wait 2-3 minutes)
3. Generate or select a profile
4. Click **ENTER OBLIVION**

---

## Troubleshooting

| Problem | Solution |
|---------|----------|
| Python not found | Install Python 3.10+ from python.org |
| PyQt6 missing | Run: `pip install PyQt6 requests` |
| No profiles | Click "GENERATE AGED PROFILE" |
| Browser won't launch | Click "INSTALL DEPENDENCIES" first |
